# 追逐打字（Chase-Type）

一款西部主题的中文拼音打字游戏。玩家在横向卷轴场景中追逐逃犯 NPC，追上后进入对决。打字速度和准确率决定一切。

## 核心概念

- **追逐阶段（Chase）**：横向卷轴跑酷，像素风西部沙漠峡谷背景。不打字则原地停止，打字才前进，打错掉速。NPC 向后投掷障碍物词条，与跑步词合一——打对同一个词既前进又避开障碍物。NPC 逃出屏幕左边缘则输。距离归零时有过渡动画进入对决。
- **转场（DuelTransition）**：距离归零后约 1 秒的过渡动画，显示"追上了！对决开始！"。
- **对决阶段（Duel）**：追上 NPC 后触发。双方共享同一个词条，各有进度条。NPC 进度条按 WPM 自动推进，玩家打字推进自己的进度条。先填满的攻击对方。各有独立血条。
- **通缉令（WantedPoster）**：击杀 NPC 后的过渡画面，展示下一个通缉犯档案。玩家点击后进入下一轮追逐。
- **游戏结束（GameOver）**：失败画面，显示重试按钮。点击从当前 NPC 重新开始。
- **状态机（StateMachine）**：管理五个游戏状态的流转。每个状态有 enter/update/exit/render 生命周期。流转路径：Chase → DuelTransition → Duel → WantedPoster → Chase（循环），任一点失败 → GameOver → Chase（重试）。

## 生命值（HP）

- **追逐 HP（chaseHP）**：追逐阶段独立血条。被障碍物击中扣血，归零则 GameOver。
- **对决 HP（duelHP）**：对决阶段独立血条。NPC 先打完则扣玩家对决 HP，归零则 GameOver。
- 击败 NPC 后两种 HP 全部恢复。

## 词条（Prompt）

中文拼音（纯字母，无音调），显示在底部输入栏。追逐阶段一个词同时用于跑步和避障。对决阶段双方共享一个词条。词库包含约 120 个词，从短到长排列。

## 障碍物（Obstacle）

NPC 向后投掷的干扰物，与跑步词合并——打对词条即可避开。

## 逃脱（Escape）

NPC 逃出屏幕左边缘，本轮失败。

## 距离指示器（Distance Meter）

屏幕上方进度条，实时反映玩家与 NPC 之间的距离。满条=追上，空条=最远。

## 技术栈

HTML/CSS + Canvas，像素风美术。Vanilla JS ES modules。西部沙漠峡谷背景。拼音纯字母。
